# Researched Knowledge Gaps

> **Priority 0 — Agents read this FIRST.**
>
> This file logs researched knowledge, corrections to outdated information,
> and external tool/API/protocol changes discovered during agent sessions.
> If you discover something that contradicts your training data or find
> updated information about an external dependency, LOG IT HERE.

## Categories
- `api` — API endpoint changes, new versions, deprecations
- `tool` — External tool updates, CLI flag changes, new features
- `protocol` — Protocol or schema changes (REST, GraphQL, WebSocket, etc.)
- `schema` — Data schema changes, new fields, type changes
- `cli` — CLI tool updates, new commands, changed flags
- `ssh` — SSH/auth/credential changes
- `webinfo` — Web resource changes, URL changes, documentation updates
- `external` — Any other external dependency change
- `correction` — Correcting known outdated knowledge from training data
